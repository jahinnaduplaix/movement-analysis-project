## 1. Project overview

This project analyses range-of-motion changes in leather workers who participated in the 6-month Ostéart prevention program.

The dataset was collected between 2023 and 2025 among 473 leather workers. According to the project context, the Ostéart program was developed as an occupational prevention program focused on upper-limb musculoskeletal disorders, including carpal tunnel syndrome.

The analysis focuses on two range-of-motion outcomes:

- wrist extension, expressed in degrees;
- forearm supination, expressed in degrees.

Each participant was measured at two timepoints:

- `T0`: baseline measurement;
- `T1`: follow-up measurement after the 6-month prevention program.

The objective was to describe within-participant changes between T0 and T1.

---

## 2. Research question

**What changes in wrist extension and forearm supination are observed after the 6-month Ostéart prevention program in leather workers?**

The wording is intentionally cautious because the study used a pre-post design without a control group.

Therefore, the analysis can describe observed within-participant changes between T0 and T1, but it cannot establish a causal effect of the prevention program.

---

## 3. Outcomes

The primary outcome variables are the within-participant change scores:

- `delta_wrist_extension_deg` = wrist_extension_deg_T1 - wrist_extension_deg_T0;
- `delta_forearm_supination_deg` = forearm_supination_deg_T1 - forearm_supination_deg_T0.

### 4.1 Data file description

The data files used in this project are stored in the `data/` folder.

## 6. Analysis workflow

The project uses two complementary analysis files:

`main.ipynb → main.Rmd → duplaix.jahinna.html`

The Python notebook must be run first because it creates the processed dataset used by the R analysis.

### Step 1: Python analysis

Run:

`main.ipynb`

The notebook starts with a setup cell that installs the required Python packages from `requirements.txt`.
This cell should be run before the package import cells.

The Python notebook performs:

- import of the raw Excel dataset;
- cleaning of participant identifiers;
- cleaning and standardization of timepoint labels;
- conversion of range-of-motion variables to numeric format;
- removal of exact duplicate rows;
- verification of the longitudinal T0/T1 structure;
- exclusion of participants without a valid paired structure;
- reshaping from long format to wide format;
- exclusion of participants with incomplete paired data;
- flagging of anatomically high values;
- computation of T1-T0 change scores;
- export of exploratory tables and figures;
- export of the final processed dataset.

The main output of the Python analysis is:

`data/processed/analysis_dataset.csv`

### Step 2: R analysis

Run:

`main.Rmd`

The R Markdown file performs:

- import of the processed dataset;
- verification of dataset dimensions and variables;
- verification of missing values;
- verification of the anatomical plausibility flag;
- normality assessment using Q-Q plots and Shapiro-Wilk tests;
- paired Wilcoxon signed-rank tests;
- sensitivity analysis excluding anatomically flagged participants;
- export of final statistical tables;
- export of the final boxplot figure;
- generation of the final HTML report.

The final report generated from R is:

`duplaix.jahinna.html`

## 8. Limitations

This project has several limitations.

First, the study used a pre-post design without a control group. Therefore, the analysis can describe observed within-participant changes between T0 and T1, but it cannot establish a causal effect of the Ostéart prevention program.

Second, the clinical relevance of small numerical changes in range of motion should be interpreted cautiously. Statistically significant changes are not necessarily clinically meaningful changes.

Third, some anatomically high values were identified during data preparation. These values were not automatically removed from the main analysis. They were flagged and assessed through a sensitivity analysis.

Fourth, the analysis was limited to participants with complete paired data at T0 and T1. Participants with incomplete data were excluded from the final complete-case dataset.
